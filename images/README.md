# Images

Структура папки:

- `daily-rebuses/` - рабочая папка с изображениями ребусов, которые сопоставляются с `data.csv` по колонке `img_url`.

Код извлечения признаков (`extract_efficientnet_features.py`) ожидает изображения в `images/daily-rebuses/`.

Пути из датасета имеют вид `/daily-rebuses/1_v1.jpg`. Резолвер ищет:

1. `images/daily-rebuses/1_v1.jpg`;
2. `images/daily-rebuses/1.jpg`, если в датасете указан вариант с суффиксом `_v1`, а в папке лежит базовое имя.

Служебные файлы macOS (`.DS_Store`) игнорируются через `.gitignore`.

Текущая сверка с `data.csv`: найдено 326 изображений из 327 строк. Не хватает файла `images/daily-rebuses/328.jpg`; для этой строки скрипты сохранят нулевой image-вектор и выведут путь в списке пропусков.
